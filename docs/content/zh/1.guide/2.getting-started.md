## 安装
使用包管理器安装下面的依赖到你的项目中。
```bash
pnpm install vue-day-calendar dayjs
```

创建一个`vue`文件，引入`vue-day-calendar`组件。
::CodePreview{componentName="DemoBase"}

```vue
<script setup lang="ts">
  import 'vue-day-calendar/style.css'
  import VueDayCalendar from 'vue-day-calendar'
</script>

<template>
  <VueDayCalendar />
</template>
```
::
